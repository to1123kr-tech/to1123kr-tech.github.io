토지이용 체크 포인트

• 지번 검색 후 중심 위치로 이동
• 필지 경계는 화면상 검토용으로 확인
• 용도지역/지구는 추후 실제 WMS 레이어명 연결 가능
• 공시지가는 연속지적도 응답의 jiga 값을 우선 표시
• 최종 행정 판단은 토지이음·공식 대장으로 재확인
